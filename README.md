## Install 
```bash
git clone https://github.com/cooljser/vimrc.git ~/.config/nvim
```

## Key Mapping
default leader key is space
|  Key   | Description |
|  ----  | ----  |
| C-s  | Save file |
| \<leader\>w  | Save file |
